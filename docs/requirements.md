# Requirements

## Must

- [x] 사용자가 샘플 종목(3~5개) 중 하나를 선택할 수 있다.
- [x] 선택한 종목의 주가 차트가 표시된다.
- [x] 차트에서 특정 날짜(포인트)를 선택할 수 있다.
- [x] 선택한 날짜의 종가·등락률·거래량 변화율이 표시된다.
- [x] 선택한 날짜에 대해 AI 분석을 요청할 수 있다 (요인·출처는 실제 DART 공시 기반, 요약·호재/유의 판단은 아직 규칙 기반 — 실제 LLM 아님, `docs/project-plan.md` M3 참고).
- [x] 분석 결과에 주요 요인(factors)이 표시된다.
- [x] 분석 결과에 출처(sources)가 표시된다 (실제 DART 공시 링크·발췌).
- [x] 분석 결과에 분석 한계(limitations)가 표시된다.

## Should

- [x] 기간 변경(1주 / 2주 / 1개월 / 전체) — `ChartToolbar`
- [ ] 이벤트 마커(실적발표, 공시 등 차트 위 마커) — 구현하지 않음. 이제 실제 DART 공시 날짜(`data/disclosures.json`)가 있어서 데이터는 준비돼 있음, UI만 아직 없음
- [ ] 분석 결과 캐싱(동일 종목/날짜 재요청 시 API 재호출 없이 표시) — 구현하지 않음
- [ ] 모바일 반응형 — 기본 레이아웃은 좁은 화면에서도 깨지지 않지만, 별도 반응형 브레이크포인트는 아직 없음

## Could

- [ ] 종목 비교(2개 이상 종목 동시 표시) — 구현하지 않음
- [ ] 분석 결과에 대한 후속 질문(대화형 follow-up) — 구현하지 않음
- [ ] 분석 기록 저장(과거 조회 이력) — 구현하지 않음, 저장소/DB가 필요하므로 범위 밖

## Explicitly excluded (요청 사항)

로그인, 회원가입, 계좌 연동, 포트폴리오, 주문, 결제, 관리자 페이지는 이번 및 향후 계획에서 제외됩니다.
